## ❓ Jawaban Pertanyaan untuk Self-Check

1. Apa perbedaan antara inline, internal, dan external CSS?

    Inline : Ditulis langsung di dalam Tag HTML

    Internal: Ditulis di dalam Tag `<style>` di dalam file HTML biasanay di simpan di Tag `<head>`

    External: Ditulis terpisah didalam file sendiri file CSS
    
2. Kapan menggunakan class selector vs ID selector?

    Class selector: digunakan untuk styling ke banyak elemen (fleksible, reusable)
    
    ID selector: digunakan untuk elemen unik (spesifik,unik)
    
3. Apa yang terjadi jika `box-sizing: border-box` tidak digunakan?

    Harus setting manual padding, width, border dan ukurannya tidah stabil layout bisa melebar tanpa sadar
    
4. Bagaimana cara center element secara horizontal?

    Menggunakan `text-align: center;`
    
5. Apa fungsi `background-size: cover`?

    Background gambar akan menurtupi semua area tanpa merusak rasio gambar
